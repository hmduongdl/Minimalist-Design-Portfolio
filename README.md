Minimalist Design Portfolio

This repository hosts a small, static personal portfolio website showcasing projects and contact information.

## Technologies

- Vite
- TypeScript
- Tailwind CSS
- PostCSS
- Framer Motion
- Deployed on Vercel

## Project structure (important parts)

- `public/` — static assets (images, icons)
- `src/` — source files
  - `src/main.ts` — app bootstrap and renderers
  - `src/style.css` — global styles
- `index.html` — main HTML template
- `package.json`, `vite.config.ts`, `vercel.json` — build & deploy config

## Quick start

1. Install dependencies:

```
npm install
```

2. Run development server:

```
npm run dev
```

3. Build for production:

```
npm run build
```
